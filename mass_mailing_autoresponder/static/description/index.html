Mail Autoresponder

Create Events and Notify your Partners

Step1: Navigate to the Customer Dialogue and click on the menu

Step 2: Create an Event

Step 3: Fill in the Appropriate fields

Step 4: Save and let the event start running

Once the event is active, then appropriate email will be sent as when due.
Note:

  • On the Email Template lists, you can set when a particular email should be sent out and the email to be sent out.
  • Once an event is ended, you can set the state to cancel and that marks the end of that event and any email that is meant to go out.
